# Mobile Track Stage 1 Task

Hello, Mobile Ninjas.
Welcome to Stage 1. This task is an extension of the Stage 0 task.

## Task Description

You are required to extend your Smart Utility Toolkit App by adding a simple task/checklist manager feature that allows users to:

- Convert units of their choice (already handled)
- Create and mark tasks as completed
- Edit and delete tasks
- Persist tasks locally

This feature should be fully integrated into your existing Stage 0 app, maintaining consistent architecture, design, and navigation.

The app must support offline functionality and include persistent local storage.

When completed, upload your APK/IPA file to Appetize and generate a public preview link for submission.

Task explainer video and Airtable links: [Video](https://www.tiktok.com/@hnginternship/video/7628599166846913814?is_from_webapp=1&web_id=7599354990642546194) and Airtable.

## Task Deliverables

- Fully functional Smart Utility Toolkit App with a working task/checklist manager feature
- Persistent offline storage
- A clean UI that is consistent with the main app and seamless navigation
- Updated GitHub repository with a well-written README.md file explaining your app, features, and how to run it
- A LinkedIn or X (Twitter) documentation post, tagging the HNG Internship account, outlining:
  - Your development process
  - Challenges encountered
  - The purpose of the task
  - How the task has improved (or will improve) your development process
