# Mobile Track Stage 0 Task

Hello, Mobile Ninjas!! 🥷🏽

Welcome to Stage 0! This stage validates your core mobile development skills, architecture discipline, and UI craftsmanship. This is not for beginners, but to show us how solid your foundation really is.

## Task Description

You are required to build a Smart Utility Toolkit Mobile App that provides a collection of essential everyday tools inside a single mobile application.

Your app should include:
- Unit Converter (Length, Temperature, Weight, Currency, etc.)

## Notes

You are encouraged to build a clean and scalable app, ensuring smooth navigation and a professional UI/UX experience.

When completed, upload your APK/IPA file to [Appetize](http://appetize.io/) and generate a public preview link for submission.

## Task Deliverables

- Functional Smart Utility Toolkit App
- At least three functional utility modules
- Clean UI and seamless navigation
